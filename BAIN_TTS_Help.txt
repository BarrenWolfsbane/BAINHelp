BAIN TTS Help File

To Change voice Type '!setvoice <ModelName>'
Some models have more then one speaker type '!setspeaker <number>' to choose a speaker
To Change pitch Type '!setpitch <0.01-3.0>'
To Change formant Type '!setformant <0.01-5.0>'
