BAIN TTS Help File

ALL commands can be used on ANY Livestreaming platform while streamer is active.
The Following are Text To Speech Commands allowing the voice that reads your text to be altered to your liking as a viewer
To Change Voice Model Type '!setvoice <ModelName>' ( Models and Commands Listed Below )
Some models have more then one speaker type '!setspeaker <number>' to choose a speaker
To Change pitch Type '!setpitch <0.01-3.0>'
To Change formant Type '!setformant <0.01-5.0>'

The Default Voicemodel is 'en-us-libritts-high' with the default speaker of '0'
